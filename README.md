# AI Tools News Monitoring Kit

English | [简体中文](README.zh-CN.md)

News monitoring workflows for tracking AI product launches, feature updates, pricing changes, and category trends.

This repository is a neutral, practical resource for AI tools news monitoring, product launch tracking, feature update review, and trend digest workflows. It is designed to help teams make clearer decisions, write better review notes, and keep workflows reusable.

## Why This Repository Exists

AI tools change quickly. A monitoring workflow helps directory editors and operators separate meaningful product updates from noise, then turn them into useful news, category refreshes, and comparison updates.

## Who This Is For

- AI directory editors tracking product updates
- content teams writing AI news digests
- operators monitoring pricing and feature changes

## Core Workflow

1. Define the review or production goal
2. Collect the required inputs and constraints
3. Use one template to make decisions comparable
4. Document fit, risks, and next actions
5. Refresh the workflow after real usage data
